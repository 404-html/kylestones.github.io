1 安装 Ubuntu18.04

  1. 从 中科大源 下载系统镜像，并制作 u 盘启动盘
  1. 启动电脑，修改为 U 盘启动
  2. 关闭 uefi 安全检查，有秘钥的通通删掉
  3. 使用 U 盘启动后，选择安装 Ubuntu
  4. 手动分区，建议只有两个分区，一个根分区，一个 home 分区；这样在需要重装系统的时候可以格式化根分区，同时保留 home 分区 的内容
  5. 安装完成后，修改 root 的密码，从而可以使用 root 登录系统

2 安装 NVIDIA 驱动

硬件配置

GPU GTX2070
  1. 由于 GPU 较新，需要添加 PPA；如非必要可以直接使用 Ubuntu repository
  2. 查看驱动列表
  3. 安装驱动
  4. 重启使驱动生效

参考 2 Ways to Install Nvidia Driver on Ubuntu 18.04 (GUI & Command Line)

3 安装 CUDA

按照文件 <CUDA_ Installation_ Guide_ Linux.pdf> 说明一步步就可以正确安装，且推荐使用 deb 安装，非常简单，强烈不建议使用 run 文件安装，有点复杂，而且我没有安装成功。不知道为什么网上的教程都是在使用 runfile 安装！@#￥%……&*

  1. 安装前检测，这个正常安装完 Ubuntu18.04 和显卡驱动，一般没有问题，但是确认一下也无妨
  2. 下载安装 cuda-repo-<distro>_<version>_<architecture>.deb
  3. 安装完成后，添加环境变量
  4. 安装例子，并测试

4 安装 cuDNN

仍然建议使用 deb 文件安装，简单方便；手册 <cuDNN-Installation-Guide.pdf>

5 安装 GPU mxnet

已经可以使用 pip 安装，所以也是 so easy

6 后记

  1. 如果有安装手册，严格按照说明安装即可
  2. 如果必要，尽量使用 Google 查看他人的安装指导
  3. 牛 X 公司的软件也是有 bug 的，尤其是较新的版本；开始使用 ppa 安装显卡驱动的时候就没有成功
  4. 新组装的电脑，系统重装了 \(N^N\) 次，感觉都要把电脑整残废了，那些不靠谱的博客
